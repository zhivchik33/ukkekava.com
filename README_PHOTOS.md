# Инструкция по добавлению фотографий в галерею "Заходи"

## Шаг 1: Подготовка фотографий

1. **Создайте папку для фотографий** (если еще не создана):
   ```
   my-project/images/events/
   ```

2. **Поместите ваши фотографии** в эту папку

3. **Рекомендации по фотографиям**:
   - Форматы: JPG, PNG, WebP
   - Размер: 800x600px или 1200x800px
   - Имена файлов: используйте понятные названия, например:
     - `culture_2023_1.jpg`
     - `social_2024_meeting.jpg`
     - `education_2025_workshop.jpg`

## Шаг 2: Добавление информации о фотографиях

Откройте файл `js/gallery.js` и найдите секцию `loadPhotos()`.

### Пример добавления фотографии:

```javascript
culture: {
    2023: [
        { 
            src: 'images/events/culture_2023_1.jpg', 
            title: 'Український культурний фестиваль', 
            description: 'Свято української культури та традицій' 
        },
        { 
            src: 'images/events/culture_2023_2.jpg', 
            title: 'Народні ремесла', 
            description: 'Мастер-клас з вишивки та народного мистецтва' 
        }
    ],
    2024: [
        // Добавьте фотографии 2024 года
    ]
}
```

### Структура данных:

```javascript
{
    src: 'путь_к_фотографии',           // Путь к файлу фотографии
    title: 'Название события',           // Заголовок события
    description: 'Описание события'      // Подробное описание
}
```

## Шаг 3: Категории событий

Доступные категории:
- `culture` - Збереження української культури та ідентичності
- `social` - Соціальна підтримка та інтеграція українців
- `education` - Освітній розвиток
- `humanitarian` - Гуманітарна місія
- `psychology` - Психологічна підтримка
- `civic` - Громадянська активність і партнерство

## Шаг 4: Годы

Для каждой категории можно добавить фотографии по годам:
- 2022
- 2023
- 2024
- 2025

## Пример полной структуры:

```javascript
this.photos = {

                src: 'images/events/social_2024_1.jpg',

```

## Шаг 5: Проверка

После добавления фотографий:
1. Сохраните файл `js/gallery.js`
2. Обновите страницу в браузере
3. Нажмите на любую категорию в секции "Заходи"
4. Проверьте, что фотографии отображаются корректно

## Советы:

- Используйте качественные фотографии
- Добавляйте информативные названия и описания
- Группируйте фотографии по годам и категориям
- Оптимизируйте размер файлов для быстрой загрузки
